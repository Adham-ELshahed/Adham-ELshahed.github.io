The Power Query language

The Power Query language (also known as M) was created specifically for data transformation and preparation tasks, through the use of formulas and expressions. Power Query is used in several Microsoft Products such as Microsoft Power BI, Microsoft Excel, and Microsoft Dataflows. These products all share the same underlying data transformation engine.

Browse Power Query functions alphabetically from the sidebar or choose a category below:

Access Data functions

Functions that enable connection to and retrieval of data from various external data sources including databases, web services, and cloud platforms.

Date and Time functions

Date and time functions help create calculations and transformations based on dates and time. Many of these functions are similar to Excel date and time functions.

Text functions

Text functions manipulate and transform string values, including operations for cleaning, formatting, and extracting data from text columns.

Table manipulation functions

These functions create, manipulate, and transform tables and their structure, including operations for filtering, joining, and reshaping data.

List functions

List functions work with list values, providing operations for creating, transforming, and aggregating lists of data.

Logical functions

Logical functions act upon expressions to return information about the values or perform conditional operations.

Number functions

Number functions perform mathematical operations and transformations on numeric values, including rounding, formatting, and conversion operations.

Record functions

Record functions work with record values, providing operations for creating, accessing, and transforming structured data records.

Type functions

Type functions provide information about data types and enable type checking and conversion operations.

URI functions

URI functions work with Uniform Resource Identifiers, providing operations for parsing and constructing URI values.

Value functions

Value functions provide operations for working with generic values, including comparison, conversion, and metadata operations.

Expression functions

Expression functions enable dynamic evaluation and manipulation of Power Query expressions and provide metadata about query execution.
